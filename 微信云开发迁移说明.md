# 微信云开发上线步骤

这份文档按零基础操作写。你照着顺序做，不要跳步骤。

## 先说明

我已经帮你把小程序改成微信云开发方式：

- 小程序 AppID 已配置：`YOUR_WECHAT_APPID`
- 后端接口已改成云函数：`cloudfunctions/api`
- 初始化演示数据云函数：`cloudfunctions/initData`
- 自动取消超时订单云函数：`cloudfunctions/cancelExpiredOrders`
- 图片上传已改成微信云存储

我不能直接替你完成最后上线，因为微信开发者工具需要登录你自己的微信小程序账号。但你按下面步骤操作即可。

## 第 1 步：打开项目

1. 打开 HBuilderX。
2. 点击左上角「文件」。
3. 点击「打开目录」。
4. 选择这个目录：

```text
D:\ProgramCode\develop\code\errand_app
```

## 第 2 步：运行到微信开发者工具

1. 在 HBuilderX 顶部菜单点击「运行」。
2. 选择「运行到小程序模拟器」。
3. 选择「微信开发者工具」。
4. 等待微信开发者工具自动打开。

如果提示 AppID，填写：

```text
YOUR_WECHAT_APPID
```

## 第 3 步：开通云开发

1. 在微信开发者工具顶部点击「云开发」。
2. 如果提示开通，点击「开通」。
3. 创建一个云开发环境。
4. 创建完成后，找到「环境 ID」。

环境 ID 一般长这样：

```text
cloud1-xxxxxx
```

注意：环境 ID 不是 AppID。

## 第 4 步：填写云环境 ID

打开这个文件：

```text
D:\ProgramCode\develop\code\errand_app\config.js
```

找到：

```js
env: '请填写你的云开发环境ID'
```

改成你的环境 ID，例如：

```js
env: 'cloud1-xxxxxx'
```

保存文件。

## 第 5 步：重新运行小程序

回到 HBuilderX：

1. 再次点击「运行」。
2. 选择「运行到小程序模拟器」。
3. 选择「微信开发者工具」。

这样微信开发者工具里的代码才会更新。

## 第 6 步：同步云函数目录

打开 Windows PowerShell。

进入项目目录：

```powershell
cd D:\ProgramCode\develop\code\errand_app
```

执行：

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\sync-cloudfunctions.ps1
```

看到类似下面内容就说明成功：

```text
Cloud functions synced to ...
cloudfunctionRoot configured in ...
```

## 第 7 步：上传云函数

回到微信开发者工具。

左侧文件列表里应该能看到：

```text
cloudfunctions
```

里面有 3 个云函数：

```text
api
initData
cancelExpiredOrders
```

分别操作：

1. 右键 `api`
2. 点击「上传并部署：云端安装依赖」
3. 等待完成

然后同样操作：

```text
initData
cancelExpiredOrders
```

三个都要上传。

## 第 8 步：初始化演示数据

在微信开发者工具里：

1. 右键 `initData`
2. 点击「云端测试」
3. 直接点击「运行」

如果返回：

```text
code: "200"
msg: "初始化完成"
```

说明数据初始化成功。

初始化后有两个演示账号：

```text
普通用户：user / demo-password
认证骑手：rider / demo-password
```

## 第 9 步：本地测试

在微信开发者工具模拟器里测试：

1. 登录 `user / demo-password`
2. 添加地址
3. 下单
4. 退出登录
5. 登录 `rider / demo-password`
6. 首页接单
7. 查看骑手订单
8. 确认送达
9. 再登录普通用户确认收货、评价

这些流程能跑通，就可以上传体验版。

## 第 10 步：上传体验版

在微信开发者工具右上角：

1. 点击「上传」。
2. 版本号可以填：

```text
1.0.0
```

3. 项目备注可以填：

```text
校园跑腿小程序云开发演示版
```

4. 点击上传。

## 第 11 步：提交审核

打开微信公众平台：

```text
https://mp.weixin.qq.com/
```

登录你的小程序账号。

然后：

1. 点击「版本管理」。
2. 找到刚上传的开发版本。
3. 设置为体验版，先用手机测试。
4. 测试没问题后点击「提交审核」。
5. 审核通过后点击「发布」。

## 常见问题

### 1. 找不到 cloudfunctions

重新执行：

```powershell
cd D:\ProgramCode\develop\code\errand_app
powershell -ExecutionPolicy Bypass -File .\tools\sync-cloudfunctions.ps1
```

### 2. 调用云函数失败

检查 `config.js` 里的环境 ID 是否填对。

### 3. 登录失败

先确认是否执行过 `initData` 云函数。

### 4. 上传云函数时报依赖错误

上传时一定要选：

```text
上传并部署：云端安装依赖
```

不要只选普通上传。

### 5. 图片上传失败

确认云开发已经开通，并且 `config.js` 环境 ID 正确。
